# Latev Engineering Services - Corporate Website

Latev Engineering Services is a modern, high-performance corporate website built for an oilfield services and engineering company specializing in coiled tubing downhole tools, well intervention, and engineering solutions.

## Overview
This project was built to give Latev Engineering a strong online presence and a polished lead-generation platform. The goal was to present the company clearly, build credibility, and make it easy for potential clients and partners to understand their services.

## Key Features
- Hero section with strong brand messaging and service highlights
- Interactive FAQ page with category filtering and accordion animations
- Contact page with form validation, map integration, and social links
- Service grid showcasing core offerings
- Testimonial carousel for client proof
- Animated performance metrics and project highlights
- Responsive navigation for all screen sizes
- Cookie consent system for compliance
- Modern contact form with validation and toast notifications

## Tech Stack
- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Tabler Icons
- React Hook Form
- Sonner
- Custom UI components

## Architecture
The project uses a modular component-based structure for maintainability and reuse.

- app, page routing and layouts
- components, reusable UI and page-specific components
- data, static content and structured page data
- types, TypeScript definitions
- styles, global styles and Tailwind setup
